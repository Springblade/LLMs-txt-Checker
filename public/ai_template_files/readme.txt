AI Discovery Files — Template Pack
https://www.ai-visibility.org.uk/specifications/

This archive contains templates for all 10 AI Discovery Files. These files
help AI systems such as ChatGPT, Claude, Gemini, and Perplexity discover,
interpret, and accurately represent your business.


GETTING STARTED
---------------

1. Open each template and replace the [placeholder] values with your
   actual business information.

2. Ensure the Canonical Identity Block (business name, brand name,
   services line) is identical across every file. Consistency is how
   AI systems build confidence in your identity.

3. Use markdown hyperlinks for all URL references in your files:
   [Link Text](https://www.example.com/page/)
   This gives AI systems anchor text context about what each link
   leads to, enabling richer citations and more accurate references.

4. Upload the completed files to your website root, e.g.:
   https://www.yourwebsite.com/llms.txt
   https://www.yourwebsite.com/ai.txt
   https://www.yourwebsite.com/identity.json
   ...and so on for each file you create.

5. You do not need to implement all 10 files. Start with llms.txt and
   ai.txt, then add more as needed. See the Quick Start Guide for a
   recommended implementation path:
   https://www.ai-visibility.org.uk/specifications/quick-start/


WHERE TO PUBLISH
----------------

All AI Discovery Files must be placed at your website root (the top-level
directory served by your web server). AI systems expect to find them at
predictable URLs like:

   https://www.yourwebsite.com/llms.txt
   https://www.yourwebsite.com/ai.json

Do not place them in subdirectories. Do not rename them.


SPECIFICATION ATTRIBUTIONS
--------------------------

Each template contains a specification reference at the end, for example:

   ---
   Specification: llms.txt (ADF-001)
   https://www.ai-visibility.org.uk/specifications/llms-txt/

You are not required to keep these attribution lines in your published
files. However, we recommend leaving them in place. Here is why:

- The AI Visibility Directory checks for these attribution links when
  determining listing eligibility. Sites with attribution qualify for a
  Full Listing, which includes a dedicated profile page with dofollow
  backlinks to your website — a genuine SEO authority signal.

- These files are not indexed by Google or other search engines. They
  are plain text and JSON files consumed by AI systems, not public-facing
  web pages. The attribution has zero impact on your SEO, positive or
  negative.

- The attribution helps AI systems trace the specification origin,
  improving the accuracy of how they interpret your files.

In short: leaving the attribution costs nothing and qualifies you for
enhanced directory visibility.


AI VISIBILITY DIRECTORY
-----------------------

Once your AI Discovery Files are published, register your website in the
AI Visibility Directory — the verified registry of websites implementing
AI Discovery Files.

The directory offers two listing tiers:

  Basic Listing (free)
  Card entry in the directory with automated file validation. Open to
  any site with a valid llms.txt file.

  Full Listing (free, requires attribution)
  Dedicated profile page with dofollow backlinks to your website — a
  genuine SEO authority signal from a topically relevant, verified
  source. Includes an attribution badge and enhanced visibility.
